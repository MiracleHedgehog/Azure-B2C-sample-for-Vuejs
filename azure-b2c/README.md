# Azure-B2C-sample

- `Azure Active Directory B2C` を利用したログインフォームサンプル。

## 準備

- `config/maslConfig.template.js`から`config/maslConfig.js`を作成し、`clientId`, `authority`を貼り付ける

## Project setup

```
yarn install
```

### Run

```
yarn run serve
```
